# 9.4 Map接口

    现实生活中，我们经常需要成对存储某些信息。比如，我们使用的微信，一个手机号只能对应一个微信账户。这就是一种成对存储的关系。

    Map就是用来存储**键(key)-值(value) 对**的。 Map类中存储的“键值对”通过键来标识，所以“键对象”不能重复。

    Map 接口的实现类有**HashMap、TreeMap、HashTable、Properties**等。
