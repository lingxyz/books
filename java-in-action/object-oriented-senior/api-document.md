## 如何下载API文档

1. 下载地址，点击进入：

   http://www.oracle.com/technetwork/java/javase/documentation/jdk8-doc-downloads-2133158.html

![图5-27 API下载界面图.png](https://www.sxt.cn/360shop/Public/admin/UEditor/20170520/1495264906753471.png)
