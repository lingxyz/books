# 4.3.2 属性(field，或者叫成员变量)

   属性用于定义该类或该类对象包含的数据或者说静态特征。属性作用范围是整个类体。

   在定义成员变量时可以对其初始化，如果不对其初始化，Java使用默认的值对其初始化。

> 表 4-2 成员变量的默认值

| 数据类型     | 默认值   |
| ------------ | -------- |
| 整型         | 0        |
| 浮点型       | 0.0      |
| 字符型       | '\u0000' |
| 布尔型       | false    |
| 所有引用类型 | null     |

**属性定义格式：**

```java
[修饰符] 属性类型 属性名 = [默认值] ;
```
