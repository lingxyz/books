# 2.7 运算符(operator) 

   计算机的最基本用途之一就是执行数学运算，作为一门计算机语言，Java也提供了一套丰富的运算符来操作变量。

| 算术运算符-二元运算符 | +，-，*，/，%                    |

| 算术运算符-一元运算符 | ++，--                           | 
| 赋值运算符            | =                                | 
| 扩展运算符            | +=，-=，*=，/=                   | 
| 关系运算符            | >，<，>=，<=，==，!=  instanceof | 
| 逻辑运算符            | &&，\|\|，!，^                   | 
| 位运算符              | &，\|，^，~ ， >>，<<，>>>       |
| 条件运算符            | ? :                              | 
| 字符串连接符          | +                                |
