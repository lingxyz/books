# 2.7.3 关系运算符

​    关系运算符用来进行比较运算，如表2-9所示。关系运算的结果是布尔值：true/false；

> 表2-9 关系运算符

| 运算符 | 含义       | 示例 |
| ------ | ---------- | ---- |
| ==     | 等于       | a==b |
| !=     | 不等于     | a!=b |
| >      | 大于       | a>b  |
| <      | 小于       | a<b  |
| >=     | 大于或等于 | a>=b |
| <=     | 小于或等于 | a<=b |

**注意事项**

- =是赋值运算符，而真正的判断两个操作数是否相等的运算符是==。
- ==、!= 是所有（基本和引用）数据类型都可以使用
- \> 、>=、 <、 <= 仅针对数值类型（byte/short/int/long, float/double。以及char）
